Proyek ini dikembangkan oleh **Tim Aether Code** dari **Universitas Merdeka Malang** untuk kompetisi **Software Engineering HIMA-RPL 2026**.

## ğŸ› ï¸ Tech Stack (Teknologi)

Website ini dibangun menggunakan teknologi *Modern Front-End* tanpa ketergantungan pada framework berat (*Lightweight*):

- **HTML5**  
  Struktur semantik untuk aksesibilitas maksimal.

  Framework CSS utility-first untuk desain UI yang presisi dan responsif.

- **Alpine.js**  
  Framework Javascript minimalis untuk menangani logika kalkulator, modal, tab, dan state loading.

- **Google Fonts**
